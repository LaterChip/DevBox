# DevBox 在线开发者工具箱
纯静态离线前端工具合集，无需后端，GitHub Pages 免费部署，完全本地计算，无广告、无数据上传。

## 功能列表
1. JSON 格式化/校验
2. 时间戳转换
3. URL/Unicode/Hex 编解码
4. Base64 文本/图片互转
5. 二维码生成导出
6. 正则表达式实时测试
7. 颜色取色色值转换
8. 高强度密码生成

## 特性
- 一键暗黑模式，系统自动跟随
- LocalStorage 保存操作历史
- 完全离线可用，dist 文件夹本地打开
- 响应式 PC/移动端适配
- 无第三方埋点、无广告

## 快速部署
1. git clone 本仓库
2. npm install && npm run build
3. 推送 GitHub 自动部署 Pages
