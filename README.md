# DevBox 开发者在线工具箱
纯静态前端工具站，**零后端、完全离线、无广告、本地计算**，一键部署 GitHub Pages。

## 功能清单
1. JSON 格式化/压缩/语法校验
2. 10/13位时间戳与日期互转
3. URL 编解码、MD5/SHA256哈希
4. Base64 文本互转
5. 自定义样式二维码生成下载
6. 正则表达式实时匹配测试
7. HEX/RGB/HSL 颜色转换取色
8. 可配置高强度随机密码生成

## 项目特性
- 一键暗黑模式，自动跟随系统主题
- LocalStorage 持久化保存每个工具操作历史
- 全部计算逻辑浏览器本地执行，不上传任何数据
- 打包 dist 文件夹双击即可离线使用，无需服务器
- 响应式布局，PC/手机自适应
- 无第三方埋点、无广告、轻量极速加载

## 技术栈
Vue3 + Vite + VueRouter + Pinia + TailwindCSS

## 快速启动
```bash
# 1. 安装依赖
npm install
# 2. 本地开发预览
npm run dev
# 3. 打包静态文件
npm run build
# 4. 本地预览打包产物
npm run preview
